# moonce

#### 项目介绍
moonce项目包含个人博客项目,是一款以Spring Boot为框架的项目,用户可以自主来部署个人博客,博客包含用户登录\发表文章\评论文章.

#### 软件架构
1. Mysql
2. Spring Boot
3. Redis
4. Vue
#### 安装教程

1. 安装依赖 JDK1.8 +
2. 安装Redis和mysql数据库
3. 创建数据库moooncedb 
4. 运行编译后的JAR包
3. 输入 localhost:8080 进行访问
